# Screenshot Capture Guide

Capture 1920x1080 screenshots from the **live production site** at `eddy.guide`.

## How to capture

1. Open Chrome DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Set dimensions to 1920x1080
4. Navigate to each URL below and capture full-page screenshot

## Required screenshots

| Filename | URL | Notes |
|----------|-----|-------|
| `home.png` | `https://eddy.guide` | Full hero + Eddy Says + featured rivers |
| `rivers-list.png` | `https://eddy.guide/rivers` | All 8 rivers with condition badges visible |
| `river-detail.png` | `https://eddy.guide/rivers/current` | Map view with access points visible |
| `float-planner.png` | `https://eddy.guide/rivers/current` | With put-in/take-out selector active |
| `gauges.png` | `https://eddy.guide/gauges` | Dashboard with sparklines visible |
| `access-point.png` | `https://eddy.guide/rivers/current/access/akers-ferry` | Facilities + amenities section |
| `share-plan.png` | `https://eddy.guide/plan/...` | Any shareable plan page |
| `ask-eddy.png` | `https://eddy.guide/chat` | Chat interface with sample messages |

## For TikTok vertical (optional)

Also capture 1080x1920 (portrait) screenshots focused on the most important UI areas of each screen. Save with `-vertical` suffix (e.g., `home-vertical.png`).
